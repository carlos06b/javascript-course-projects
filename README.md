# Projetos de JavaScript

Este repositório reúne projetos práticos desenvolvidos durante meu curso de JavaScript.

## Projetos

### Calculadora
Calculadora simples para realizar operações matemáticas básicas.

### Calculadora Tip
Calculadora de gorjeta que ajuda a calcular o valor da gorjeta e o total da conta.

### Guess
Jogo de adivinhação onde o usuário tenta descobrir um número.

### Pick a Color
Aplicação para escolher ou gerar cores de forma interativa.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Objetivo

Praticar conceitos fundamentais de JavaScript, manipulação do DOM, eventos e lógica de programação.
